Conceptos Fundamentales

Variables y Tipos de Datos

JavaScript es dinámicamente tipado. Aprende sobre let, const, var y los tipos primitivos.

let numero = 42;
const texto = "Hola Mundo";
let booleano = true;
let arreglo = [1, 2, 3];
let objeto = { nombre: "JS" };

Funciones y Arrow Functions

Las funciones son bloques de código reutilizables. ES6 introdujo las arrow functions.

function saludar(nombre) {
  return `Hola, ${nombre}!`;
}

const sumar = (a, b) => a + b;
const resultado = sumar(5, 3);

Arrays y Métodos

Los arrays son estructuras de datos fundamentales con métodos poderosos.

const numeros = [1, 2, 3, 4, 5];

const dobles = numeros.map(n => n * 2);
const pares = numeros.filter(n => n % 2 === 0);
const suma = numeros.reduce((a, b) => a + b);

Objetos y Destructuring

Los objetos almacenan colecciones de pares clave-valor y soportan destructuring.

const persona = {
  nombre: "Ana",
  edad: 25,
  ciudad: "Madrid"
};

const { nombre, edad } = persona;
const copia = { ...persona, edad: 26 };

Promesas y Async/Await

Maneja operaciones asíncronas de manera elegante con promesas y async/await.

async function obtenerDatos() {
  try {
    const response = await fetch('/api/data');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error(error);
  }
}

Manipulación del DOM

Interactúa con elementos HTML usando JavaScript para crear experiencias dinámicas.

const elemento = document.querySelector('.clase');
elemento.addEventListener('click', () => {
  elemento.classList.toggle('activo');
  elemento.textContent = '¡Clickeado!';
});

Ejemplos Avanzados

Closures

function contador() {
  let count = 0;
  return function() {
    count++;
    return count;
  };
}

const miContador = contador();
console.log(miContador()); // 1
console.log(miContador()); // 2

Event Loop

setTimeout(() => {
  console.log('2');

Promise.resolve()
  .then(() => console.log('3'));

console.log('4');
// Output: 1, 4, 3, 2

Debounce Pattern

function debounce(func, delay) {
  let timeoutId;
  return function(...args) {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => {
      func.apply(this, args);
    }, delay);
  };
}
